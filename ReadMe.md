# VRCToolBox
VRChatユーザー向けに、色んな機能を詰め込む予定のアプリです。  
MITライセンスの下公開しています（使用ライブラリはそれぞれのライセンスが適用されます）。

動作保証・サポート等はいたしかねます。  
また、セキュリティリスク含めて自己責任の下ご利用ください。

インストール、アンインストール、使用方法については[GitHub Wiki][def]から参照してください。  
※インストール後、必ず「設定」をご確認ください。  
　既定ではCドライブのAppDataフォルダなどに各種データを保存するため、容量を圧迫する可能性があります。  
　また、ログ保存先などを途中で変更すると、予期せぬ動作の原因となる可能性があります。  

[def]: https://github.com/MinadukiSekina/VRCToolBox/wiki
